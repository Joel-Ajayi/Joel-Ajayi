## Currently building

**[RRQ](https://github.com/Joel-Ajayi/river-rust-queue)**, a payment processing core in Go. A closed-loop, double-entry ledger on Postgres where every transfer is one serializable transaction, events ride a transactional outbox into Kafka, and nine correctness invariants hold across six services. Rust port planned.

## Selected projects

- **[yot-run](https://github.com/Joel-Ajayi/yot-run)**: a custom async runtime in Rust. Work-stealing scheduler, epoll/kqueue reactor, lock-free task lifecycle.
- **[SmartAR Tailor](https://github.com/Techspire-V0/SmartAR_Tailor)**: 360-degree phone scan to 3D body reconstruction and ISO 8559 measurements (Gaussian Splatting, PyTorch3D, Go API).
- **[MarketPulse](https://github.com/Joel-Ajayi/cls-marketpulse)**: cross-platform inflation-tracking app, React Native (Expo) with a Rust backend.
